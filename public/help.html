How to use Movie Finder?



If you want to use Movie Finder, you can very simply navigate to the home page and use the search bar to search up any movie or TV show that you'd like to find a source for. On the following results page after search, you can click the button that says "I watched!" and get that media listed on the homepage.



If you want to find/host the project yourself, navigate to the Github and follow the instructions therein to deploy it using Nodejs.
